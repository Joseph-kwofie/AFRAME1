3D A-Frame Table Scene Project 
Project Overview
This project is a simple 3D Computer Graphics scene built using A-Frame, rendered inside a web browser.
It contains:
-	A table
-	Two chairs
-	A Milo tin
-	One milk tin
-	Two cups
-	Beautiful decorations and lighting
-	Orbit controls + Zoom control
How to Run the Project
1.	Install VS Code
2.	Install the Live Server extension
3.	Place the project files in a folder
4.	Right click the index.html→ Open with Live Server and the 3D scene is now been displayed in the browser.
Technologies Used

- Mouse

Troubleshooting
Scene not showing?
-	Make sure Live Server is running
-	Check file paths for textures
-	Open console (F12 → Console) for errors
